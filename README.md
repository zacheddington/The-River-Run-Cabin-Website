# The River Run Cabin Website

A modern, responsive website for The River Run Cabin rental property in southeast Idaho, built with static HTML, CSS, and JavaScript. Features include:

- Home page with hero image, property description, and dynamic gallery
- Area page with local attractions and links
- Q&A page with tables for rentals, restaurants, airports, and fly shops
- Contact and Book pages
- Mobile-first, fully responsive design
- No cookies, trackers, or popups
- Designed for easy deployment to GitHub Pages

## Quick Start
1. Clone this repository
2. Open `index.html` in your browser, or deploy to GitHub Pages

## Deployment
- Push to a GitHub repository
- Enable GitHub Pages in repository settings (set source to `main` branch or `/docs` folder)
- Your site will be live at `https://yourusername.github.io/reponame/`
